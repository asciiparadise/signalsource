# Time Signature

A time signature is usually treated as a technical thing.

How many beats in a bar. What kind of note gets the beat. 4/4, 3/4, 5/4, 7/8.

But emotionally, it is much more than that.

A time signature is the architecture of movement. It tells the body how to expect the next step.

## The Familiar Square

4/4 feels normal to most modern listeners because we have lived inside it for so long.

It is square, stable, symmetrical. Dance music, rock, pop, hip-hop, electronic music — so much of modern listening is built on that grid.

That stability can be powerful. It can create trance, drive and collective motion.

But because it is so familiar, we often stop noticing it.

## Odd Meters

Odd meters make counting visible again.

5/4 feels slightly extended. 7/8 often feels like a missing or shortened step. 9/8 can swing or spiral depending on how it is grouped. 11/8 can feel unstable in a way that is difficult to explain without moving your body.

That is the key point.

Odd meters are not only counted.

They are felt as imbalance, skip, lean, stumble or ritual.

## Grouping

The same time signature can feel completely different depending on grouping.

7/8 might be:

- 2 + 2 + 3
- 3 + 2 + 2
- 2 + 3 + 2

Each grouping creates a different body-feel.

That is where mathematics becomes physical.

## Signal Source Use

Time signatures matter for Signal Source because they can make music feel slightly misaligned without making it obviously chaotic.

A track in 5/4 or 7/8 can create the feeling that reality is still organised, but not in the expected way.

That is a very useful emotional state.

It connects to unease, ritual, hidden systems and altered perception.
