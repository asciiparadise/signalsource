# Apocalypse as Overlay

## Type
Theme

## Core Idea

The surface layer appears to be cults, apocalypse, religion, conspiracies or occult symbolism.

Underneath lies something more human: grief, memory, family, mortality and longing.

## Examples

### Geogaddi
Surface: numerology, hidden messages, cult imagery.

Core: childhood, wonder, fear.

### Inferno
Surface: ritual, fire, apocalypse.

Core: heartbeat, family films, memory, mortality.

### Twin Peaks: The Return
Surface: cosmic evil, portals, mystery.

Core: grief, loss, time.
