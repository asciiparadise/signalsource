# Memory

## Type
Theme

## Core Idea

Memory is not only recollection. It is distortion, decay, repetition, loss and reconstruction.

In this vault, memory connects personal experience, analogue media, childhood television, ruins, grief, loops, ghosts, degraded signals and works that feel half-remembered before they are fully understood.

## Why It Matters

Many of the strongest works here seem to operate like memories rather than stories. They do not explain themselves in a straight line. They return in fragments.
