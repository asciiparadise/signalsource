# Straight Structure - Target Taxonomy

## Purpose

This note defines the cleaner folder logic for Signal Source.

The vault has grown organically, which was useful at first, but the structure now needs to become more literal and predictable.

The rule should be simple:

People and artists go under **Artists**.

Individual works go under **Works**.

Recurring ideas go under **Concepts**.

Recurring images and objects go under **Symbol Index**.

Knowledge fields go under **Fields**.

Historical / occult / conspiracy material goes under **Hidden Systems**.

# Migration Rule

Do not move everything blindly.

Move in batches:

1. Create clean index pages first.
2. Move one domain at a time.
3. After moving, update hub links.
4. Avoid duplicate note titles unless necessary.
5. If a note is only a placeholder, mark it clearly.

The goal is not only tidiness.

The goal is trust.
