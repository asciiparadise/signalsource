# Open Threads and Missing Notes

## Purpose

This is the working cleanup checklist for Signal Source.

The vault has grown past the first expansion phase. The main task now is not to add more everything. The task is to make the existing structure trustworthy.

A note should not only sound good.

It should have context, status, links and a reason to exist.

# Rule Going Forward

For substantial notes:

```markdown
## Why It Belongs Here

## What We Actually Know

## Signal Source Reading

## Connections
```

For disputed or archive material:

```markdown
## Factual Status / Evidence Status

## What We Actually Know

## What Is Mythology
```

For symbols:

```markdown
## Core Meaning

## Where It Appears

## Signal Source Reading
```

The reflective style stays.

But the vault needs bones.
