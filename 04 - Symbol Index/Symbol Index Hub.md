# Symbol Index Hub

The Symbol Index should become one of the most useful parts of Signal Source.

Not because symbols have fixed meanings. They do not. A symbol changes depending on where it appears. Fire in *Twin Peaks* is not the same as fire in *Inferno*. A road in *Lost Highway* is not the same as a road in *Trans Canada Highway*.

The point is to track repetition.

When an image keeps appearing across unrelated works, it starts to behave like a hidden system.

## Cleanup Rule

A symbol note should answer three things:

```markdown
## Core Meaning

## Where It Appears

## Signal Source Reading
```

If it cannot answer those yet, it should stay listed as planned rather than pretending to be complete.
